# CarbonSense AI - Frontend Dashboard

A professional React dashboard for interacting with the carbon-aware AI system.

## Features

- **Interactive Chat**: Features real-time badges for **⚡ Compression**, **🌎 Region**, and **🍃 Carbon Saved**.
- **Live Metrics Dashboard**: Visualizes response latency, model distribution, and cumulative impact.
- **Dynamic Feedback**: Star-rating system integrated with Backend RL.
- **Optimized UI**: Built with Tailwind CSS and Framer Motion for premium feel.
- **Backend Sync**: Hardcoded 127.0.0.1 connectivity for local reliability.

## Tech Stack

- **React** (Vite)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Recharts** (Data Visualization)
- **Lucide React** (Icons)

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Navigate to `http://localhost:5173`.

## Configuration

- API connection simulated in `src/services/api.js`.
- Customize colors in `tailwind.config.js`.
